# Ignite Shoes App

## Deep Linking

**Listagem links**

```bash
npx uri-scheme list
```

**Abrir app usando o link:**

```bash
npx uri-scheme open com.vitorsemidio.igniteshoesapp://192.168.11.11:8081 --android
```

**Abrir app usando deep linking:**

```bash
npx uri-scheme open com.vitorsemidio.igniteshoesapp://details/7 --android
```
